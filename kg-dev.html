WEB DEVELOPER

2015 - 2017 - Kurt Geiger

As a developer at Kurt Geiger my primary focus was developing the in store application using magento.

My main duties involved:

  • Planning and implementing new features
  • Developing web applications for use in retail stores, working closely with UX, front-end and management team members
  • Developing microservices using NodeJS
  • Bug fixing
  • Testing both unit and e2e testing
  • Troubleshooting application problems with employees in retail stores
  • Building and integrating with Restful API's
  • Completing work on time to deadlines set by the business which often need a quick turnaround due to the retail nature of the company
